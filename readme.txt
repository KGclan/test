Это проект на чистом Django , я делал его в прошлом семестре для экзамена по web - программированию.Верстку я взял стандартную бутстраповскую
и переделывал под проект , для экономии времени.Тема проекта - интернет магазин мужской одежды.
1.В этом проекте был реазизован поиск товаров по названию.
2.Сортировка по категориям.
3.Сортировка по подкатегориям.
4.Динамические категории и подкатегории.
5.Возможность добавления товара.
6.Корзина.
8.Админка с фильтрами и поиском.
9.Для регистрации и авторизации использовался django allauth.
Виртуальное окружение - папка sand.
